贪吃蛇游戏 - AI智能助手版
运行说明文档

=====================================
一、运行环境要求
=====================================

1. 操作系统：Windows 10/11 (64位)
2. .NET运行时：.NET 6.0 Runtime 或更高版本
3. 内存：至少 512MB 可用内存
4. 磁盘空间：至少 100MB 可用空间
5. 网络连接：可选（用于AI在线功能）

=====================================
二、安装和启动
=====================================

方法一：命令行运行（推荐）
1. 下载并解压项目文件到任意目录
2. 打开命令提示符(CMD)或PowerShell
3. 导航到项目目录：cd "项目路径"
4. 执行构建命令：dotnet build
5. 执行运行命令：dotnet run

方法二：Visual Studio运行
1. 安装Visual Studio 2022（社区版免费）
2. 打开SnakeGameAI.csproj项目文件
3. 按F5键或点击"开始调试"按钮

方法三：直接运行可执行文件
1. 构建项目后，在bin/Debug/net6.0-windows/目录下
2. 双击SnakeGameAI.exe文件启动游戏

=====================================
三、操作说明
=====================================

基本控制：
- 方向键 ↑↓←→：控制蛇的移动方向
- 空格键(Space)：暂停/继续游戏
- R键：重新开始游戏
- ESC键：退出游戏

界面按钮：
- [开始游戏]：开始新的游戏
- [暂停]：暂停当前游戏
- [重新开始]：重置游戏到初始状态
- [获取游戏建议]：请求AI分析当前状态并提供建议
- [游戏技巧]：获取AI提供的游戏技巧和策略

=====================================
四、游戏规则
=====================================

1. 控制蛇移动吃食物（红色圆点）
2. 每吃一个食物，蛇身增长，分数增加10分
3. 蛇撞到墙壁或自己的身体时游戏结束
4. 随着分数增加，游戏速度会逐渐加快
5. 目标是获得尽可能高的分数

=====================================
五、AI功能说明
=====================================

智能建议功能：
- 点击"获取游戏建议"按钮
- AI会分析当前游戏状态（蛇的位置、食物位置、移动方向等）
- 提供最佳移动策略和风险警告
- 建议包括路径规划和危险预判

游戏技巧功能：
- 点击"游戏技巧"按钮
- AI提供专业的游戏策略和技巧
- 包括高级移动技巧、空间管理等
- 帮助提高游戏水平

表现分析功能：
- 游戏结束后自动触发
- AI分析本局表现与历史成绩对比
- 提供改进建议和技能评估

=====================================
七、故障排除
=====================================

问题1：程序无法启动
解决方案：
- 检查是否安装了.NET 6.0 Runtime
- 从Microsoft官网下载并安装最新版本
- 确认Windows版本兼容性

问题2：AI功能不可用
解决方案：
- 检查网络连接是否正常
- AI功能会自动降级到本地算法
- 查看程序输出的错误信息

问题3：游戏运行缓慢
解决方案：
- 关闭其他占用资源的程序
- 检查系统内存使用情况
- 在游戏设置中调整游戏速度

问题4：界面显示异常
解决方案：
- 检查屏幕分辨率设置
- 确认显示缩放比例为100%
- 重启程序或重新构建项目
